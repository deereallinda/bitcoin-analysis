#  Bitcoin Price Analysis (2013–2017)

**By Linda Mthembu - Junior Data Analyst | Python • Power BI • SQL | Aspiring Data Engineer**  

A full exploratory data analysis (EDA) of Bitcoin historical price data from **April 2013 to July 2017**.

---

##  Project Overview  

This project explores Bitcoin's behaviour over 4+ years using time-series analysis, data preprocessing, and visualisation.  
The dataset includes daily Open, High, Low, Close (OHLC), Volume, and Market Cap.

---

##  Objectives  

- Import and clean Bitcoin OHLC dataset  
- Convert `Date` into valid datetime objects  
- Visualise long-term price trends  
- Build candlestick charts with Plotly  
- Compare linear vs logarithmic closing prices  
- Perform yearly, quarterly, and monthly resampling  
- Analyse daily volatility using percentage change  

---

##  Tools & Technologies  

- **Python**: Pandas, NumPy, Matplotlib, Seaborn, Plotly  
- **Techniques**: Time-Series Analysis, Data Cleaning, Resampling, Visualization  

---

##  Key Visuals  

- OHLC trend plots  
- Candlestick chart (Plotly)  
- Linear vs log-scale price comparison  
- Yearly / quarterly / monthly average closing prices  
- Daily % change volatility graph
